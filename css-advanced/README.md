# Smile School Website

A responsive and modern educational landing page for Smile School — built with semantic HTML5 and custom CSS. This project aims to showcase clean layout structure, accessibility, and styling best practices.

## Project Overview

Smile School is a fictional platform that teaches the art of smiling through tutorials, mentorships, and community engagement. This website introduces the platform, highlights tutorials, showcases instructors, and invites users to register.


## Goals

- Clean, semantic HTML structure
- Responsive and accessible CSS styling
- Component-based layout using modern techniques (Flexbox, Grid)
- Smooth user experience on all devices

## Completed

- HTML structure with:
  - Header & navigation
  - Hero section
  - Testimonials & quotes
  - Tutorials grid
  - Membership CTA
  - FAQ section
  - Footer with social links
- Semantic tags and ARIA improvements
